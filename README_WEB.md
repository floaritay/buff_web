# BUFF自动化工具 - Web界面

这是一个整合了三个BUFF自动化脚本的Web应用，提供友好的图形界面来控制脚本运行。

## 功能特性

- 🎨 **涂鸦购买脚本**：自动筛选并购买低价涂鸦饰品
- 🔍 **Austin挂件搜枪**：搜索带有Austin挂件的低价枪械
- 🔍 **Budapest挂件搜枪**：搜索带有Budapest挂件的低价枪械
- 📊 **实时任务监控**：查看脚本运行日志和状态
- 🚀 **后台运行**：脚本在后台线程中运行，不阻塞界面
- 📜 **历史记录**：查看所有历史任务执行记录
- ⏰ **定时任务**：设置定时任务自动执行脚本
- 🌙 **主题切换**：支持白天/黑夜模式切换
- 🗑️ **清空功能**：支持清空任务列表和历史记录
- 🎯 **标签页导航**：清晰的功能分区

## 安装步骤

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行Web应用：
```bash
python app.py
```

3. 在浏览器中访问：
```
http://localhost:5000
```

## 使用说明

### 通用设置

- **Cookie**：可选输入，留空则使用已保存的cookie.txt
- 如果是首次使用，需要先在浏览器中获取BUFF网站的Cookie并粘贴

### 涂鸦购买

- **最大价格**：设置购买的最高价格（默认0.05元）
- **最大购买数量**：最多购买多少个饰品（默认10个）
- 支持数字按钮快速调整参数
- 可以清空该类型的所有任务

### 挂件搜枪（Austin/Budapest）

- **最大价格**：设置购买的最高价格（默认0.3元）
- **最大页数**：遍历多少页挂件（默认18/15页）
- **最大购买数量**：最多购买多少个饰品（默认10个）
- 支持数字按钮快速调整参数
- 可以清空该类型的所有任务

### 历史记录

- 查看所有已完成任务的执行记录
- 包含任务输出日志
- 支持清空所有历史记录

### 定时任务

- **任务类型**：选择要定时执行的脚本类型
- **常用示例**：提供多个常用的 cron 表达式示例
  - 每小时执行一次
  - 每2小时执行一次
  - 每30分钟执行一次
  - 每天早上9点执行
  - 每天下午6点执行
  - 每周一早上8点30分
- **Cron表达式**：自定义 cron 表达式，可以修改示例中的数字
- 定时任务执行后会自动保存到历史记录
- 显示下次执行和上次执行时间

### 主题切换

- 点击右上角的主题切换按钮
- 支持白天模式（☀️）和黑夜模式（🌙）
- 主题选择会自动保存，下次访问保持不变

## 文件说明

- `app.py` - Flask Web应用主文件
- `templates/index.html` - 前端界面
- `requirements.txt` - Python依赖包
- `buff_buyer.py` - 涂鸦购买脚本（原有）
- `buff_charm_searcher_austin.py` - Austin挂件搜枪脚本（原有）
- `buff_charm_searcher_budapest.py` - Budapest挂件搜枪脚本（原有）
- `task_history.json` - 历史任务记录（自动生成）

## Cron表达式说明

标准的 cron 表达式格式为：`分 时 日 月 周`

| 字段 | 允许值 | 特殊字符 |
|------|--------|----------|
| 分 | 0-59 | , - * / |
| 时 | 0-23 | , - * / |
| 日 | 1-31 | , - * / ? L W |
| 月 | 1-12 或 JAN-DEC | , - * / |
| 周 | 0-6 或 SUN-SAT（0=周日） | , - * / ? L # |

常用示例：
- `0 * * * *` - 每小时执行
- `0 */2 * * *` - 每2小时执行
- `0 9 * * *` - 每天9点执行
- `0 18 * * *` - 每天18点执行
- `0 */30 * * *` - 每30分钟执行
- `30 8 * * 1` - 每周一8点30分执行

## 注意事项

1. 确保原有三个脚本文件在同一目录下
2. Cookie会自动保存到cookie.txt文件
3. 已尝试购买的商品记录会自动保存
4. 脚本运行时间可能较长，请耐心等待
5. 可以同时运行多个任务
6. 定时任务需要保持应用运行才能执行
7. 定时任务执行间隔为30秒检查一次
